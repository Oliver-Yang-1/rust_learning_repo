# 练习：数据类型

来源章节：`010-1-4-2-数据类型.md`

## 练习目标

练习以下概念：

- 类型标注
- 整数、浮点数、布尔、字符
- 元组与解构
- 数组与索引访问

## 要求

1. 把字符串 `"42"` 解析成 `u32`
2. 定义一个 `f64` 浮点数
3. 定义一个 `bool`
4. 定义一个 `char`
5. 创建元组 `(500, 6.4, 1)`，取出第二个值
6. 创建数组 `[10, 20, 30, 40, 50]`，打印第一个和最后一个元素

## 目标输出

```txt
guess: 42
temperature: 36.5
is_rust_fun: true
initial: R
tuple second: 6.4
array first: 10
array last: 50
```

## 提示

- `parse()` 常常需要类型标注
- `char` 用单引号
- 数组索引从 `0` 开始
