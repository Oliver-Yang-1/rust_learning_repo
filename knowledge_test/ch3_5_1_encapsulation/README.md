# 练习：封装

来源章节：`081-3-5-1-面向对象语言的特点.md`

## 练习目标

练习以下概念：

- 封装
- 私有字段
- 公有方法
- 通过 API 保持内部状态一致

## 要求

1. 实现 `AveragedCollection`
2. 让 `list` 和 `average` 保持私有
3. 通过 `add` 和 `remove` 更新集合
4. 通过 `average()` 读取缓存平均值
5. 用测试验证平均值不会和内部列表失同步

## 运行方式

```bash
cargo test
```

## 目标行为

- 添加 `10, 20, 30` 后平均值应为 `20.0`
- 删除 `30` 后平均值应为 `15.0`
- 删除最后一个元素后平均值应重置为 `0.0`

## 提示

- 重点不是“能算平均值”，而是“外部代码不能绕过 API 破坏平均值缓存”
- 这正是封装在 Rust 里的典型体现：类型公开，字段默认私有
