# 练习：运算符与符号

来源附录：

- 附录 B：运算符与符号

## 练习目标

练习以下概念：

- 算术与赋值运算符
- 范围与切片
- 比较与逻辑运算
- turbofish `::<...>`
- `?` 错误传播

## 要求

1. 使用 `+=`
2. 使用切片范围 `1..4`
3. 使用 `>`、`<`、`&&`
4. 用 `parse::<i32>()` 展示 turbofish
5. 在函数中使用 `?`

## 目标输出

```txt
operator += : 15
range slice: [2, 3, 4]
logic compare: true
turbofish parse: 42
question mark demo: 10
```

## 提示

- 这题不是覆盖附录 B 的全部表格，而是挑最常见、最值得上手的几个符号来练
