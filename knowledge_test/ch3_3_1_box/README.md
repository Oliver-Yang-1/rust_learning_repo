# 练习：`Box<T>`

来源章节：`069-3-3-1-Box-指向堆上数据,并且可确定大小.md`

## 练习目标

练习以下概念：

- `Box::new`
- 堆上分配
- 递归类型
- 用 `Box<T>` 让递归类型拥有已知大小

## 要求

1. 把一个整数放进 `Box<T>`
2. 定义一个递归链表 `List`
3. 使用 `Box<List>` 让链表可以编译
4. 递归计算链表 `1 -> 2 -> 3` 的总和

## 目标输出

```txt
boxed number: 5
list sum: 6
```

## 提示

- 直接在枚举里放下一个同类型值会导致“无限大小”问题
- `Box<T>` 本身大小固定，所以很适合打断递归类型
