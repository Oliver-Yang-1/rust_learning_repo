# 练习：vector

来源章节：`034-2-2-1-vector.md`

## 练习目标

练习以下概念：

- 创建 `Vec`
- `push` 增加元素
- `get` 安全访问元素
- 遍历元素
- 通过可变引用修改元素

## 要求

1. 创建一个空的 `Vec<i32>`
2. 依次 `push` 10、20、30
3. 用 `get(1)` 读取第二个元素并打印
4. 遍历所有元素，打印它们的总和
5. 再遍历一次，把每个元素都加 5
6. 打印修改后的 vector

## 目标输出

```txt
second item: 20
sum: 60
updated vector: [15, 25, 35]
```

## 提示

- `get` 返回 `Option<&T>`
- 修改元素时要遍历 `&mut v`
- 修改可变引用指向的值时要用 `*item += 5`
