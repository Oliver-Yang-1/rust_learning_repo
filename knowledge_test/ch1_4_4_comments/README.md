# 练习：注释

来源章节：`012-1-4-4-注释.md`

## 练习目标

练习以下概念：

- 单行注释 `//`
- 多行连续注释
- 行尾注释
- 在合适的位置解释代码意图

## 要求

请在 `src/main.rs` 中完成下面的注释任务：

1. 在文件顶部补一段 2 到 3 行注释，说明程序用途
2. 在折扣计算逻辑前补一段注释，解释为什么学生有折扣
3. 给某一行变量定义添加一个行尾注释
4. 保持程序输出不变

## 目标输出

```txt
price: 100
final price: 80
tag: student
```

## 提示

- 注释不会影响编译结果
- 好注释解释“为什么”，不是机械复述“做了什么”
