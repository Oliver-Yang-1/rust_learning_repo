# 练习：常见集合总览

来源章节：`033-2-2-常见集合.md`

## 练习目标

练习以下概念：

- `Vec<T>`
- `String`
- `HashMap<K, V>`
- 不同集合的适用场景

## 要求

1. 用 `Vec<i32>` 保存三个分数
2. 计算并打印总和
3. 用 `String` 拼接一句课程提示
4. 用 `HashMap<String, i32>` 保存两个队伍的分数
5. 读取并打印 `Blue` 队分数

## 目标输出

```txt
score sum: 270
message: Rust collections
blue score: 10
```

## 提示

- `Vec` 适合一组同类型有序值
- `String` 适合可增长文本
- `HashMap` 适合键值映射
