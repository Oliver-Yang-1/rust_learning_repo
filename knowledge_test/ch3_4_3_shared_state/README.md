# 练习：共享状态并发

来源章节：`078-3-4-3-共享状态.md`

## 练习目标

练习以下概念：

- `Mutex<T>`
- `Arc<T>`
- 在线程间共享状态
- 自动释放锁

## 要求

1. 用 `Arc<Mutex<i32>>` 创建一个共享计数器
2. 启动 10 个线程
3. 每个线程获取锁后把计数器加一
4. 等待所有线程结束
5. 打印最终结果

## 目标输出

```txt
final counter: 10
```

## 提示

- `Mutex<T>` 负责一次只允许一个线程修改数据
- `Arc<T>` 负责让多个线程安全地共享所有权
- 锁会随着 guard 离开作用域自动释放
