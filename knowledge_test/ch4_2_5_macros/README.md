# 练习：宏

来源章节：`094-4-2-5-宏.md`

## 练习目标

练习以下概念：

- `macro_rules!`
- 可变参数
- 宏展开生成重复代码
- 宏与函数的区别

## 要求

1. 实现一个简化版 `simple_vec!` 宏
2. 让它接受可变数量参数
3. 再实现一个 `repeat_message!` 宏
4. 运行程序，观察宏调用展开后的效果

## 目标输出

```txt
macro vec len: 3
macro repeat: rust, rust, rust
```

## 提示

- 宏是在编译前展开的“生成代码的代码”
- 这题先专注 `macro_rules!`，不要求你现在就搭完整过程宏工程
