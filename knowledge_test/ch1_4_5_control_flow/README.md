# 练习：控制流

来源章节：`013-1-4-5-控制流.md`

## 练习目标

练习以下概念：

- `if / else if / else`
- 在 `let` 中使用 `if`
- `loop` 和 `break`
- `while`
- `for`

## 要求

1. 定义分数 `score = 92`
2. 用 `if / else if / else` 生成等级 `A / B / C`
3. 用 `if` 表达式生成 `status`
4. 用 `loop` 让计数器加到 3，并通过 `break` 返回 `6`
5. 用 `while` 打印从 3 到 1 的倒计时
6. 用 `for` 遍历数组 `[10, 20, 30]`

## 目标输出

```txt
grade: A
status: pass
loop result: 6
countdown:
3
2
1
values:
10
20
30
```

## 提示

- `if` 的各个分支返回值类型必须一致
- `loop` 可以配合 `break value`
- 遍历数组时优先使用 `for`
