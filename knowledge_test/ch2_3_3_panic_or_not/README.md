# 练习：panic! 还是不 panic!

来源章节：`040-2-3-3-panic!-还是不-panic!.md`

## 练习目标

练习以下概念：

- 什么时候返回 `Result`
- 什么时候使用 `panic!`
- 用自定义类型保证数据有效性
- 通过构造函数检查不变量

## 要求

1. 定义结构体 `Guess`，内部保存 `i32`
2. 写关联函数 `Guess::new(value: i32) -> Guess`
3. 当 `value` 不在 `1..=100` 范围内时 `panic!`
4. 写方法 `value(&self) -> i32`
5. 在 `main` 中创建一个合法的 `Guess`
6. 打印该值

## 目标输出

```txt
validated guess: 42
```

## 提示

- 这里的重点是把“值必须合法”编码进类型本身
- 非法值代表调用方违反了契约，因此 panic! 是合理的
- 你可以把 `42` 改成 `142`，观察 panic 行为
