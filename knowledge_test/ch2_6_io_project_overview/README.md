# 练习：I/O 项目总览

来源章节：`049-2-6-一个-I/O-项目-构建命令行程序.md`

## 练习目标

练习以下概念：

- 命令行参数
- 文件读取
- 简单搜索
- 命令行工具的输入与输出

## 要求

1. 让程序接收两个命令行参数：查询词和文件名
2. 读取指定文件内容
3. 打印查询词、文件名
4. 统计文件中包含查询词的行数

## 运行方式

```bash
cargo run body poem.txt
```

## 目标输出

```txt
query: body
file: poem.txt
matched lines: 3
```

## 提示

- 可以用 `std::env::args()` 读取参数
- 可以用 `std::fs::read_to_string()` 读取文件
- 先把功能做小做通，再逐步重构
