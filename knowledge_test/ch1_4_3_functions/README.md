# 练习：函数

来源章节：`011-1-4-3-函数如何工作.md`

## 练习目标

练习以下概念：

- 定义函数
- 函数参数
- 代码块表达式
- 返回值

## 要求

1. 实现 `print_profile(name: &str, age: u8)`，打印名字和年龄
2. 实现 `plus_one(x: i32) -> i32`
3. 实现 `lucky_number() -> i32`
4. 在 `main` 中写一个代码块表达式，把结果绑定给 `block_result`

## 目标输出

```txt
name: Ferris, age: 18
next: 6
lucky: 7
block result: 4
```

## 提示

- 返回值表达式最后一行不要加分号
- 代码块也可以返回值
