# 练习：模式的全部语法

来源章节：`088-4-1-3-模式的全部语法.md`

## 练习目标

练习以下概念：

- 字面值模式
- 范围模式
- 结构体解构
- `_` 忽略值
- `..` 忽略剩余值
- 匹配守卫
- `@` 绑定

## 要求

1. 用范围模式匹配 `1..=5`
2. 解构 `Point`
3. 用 `_` 忽略元组中的部分值
4. 用 `..` 只取第一个和最后一个值
5. 用匹配守卫添加额外条件
6. 用 `@` 在匹配范围时保存匹配到的值

## 目标输出

```txt
range pattern: one through five
struct pattern: on y axis at 7
ignore pattern: 2, 8, 32
dotdot pattern: 2, 32
guard pattern: less than five
at pattern: 5
```

## 提示

- 这一节更像“模式语法清单”，适合边运行边回忆每种语法做什么
- `@` 是很容易忘的语法点：它能让你“既匹配又绑定”
